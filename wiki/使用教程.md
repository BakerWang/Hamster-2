# 安装方法  
1. 安装环境：
>* JDK7以上
>* Tomcat8以上
>* MySQL 5.5以上
比这些版本低的本人没试验过，有兴趣你可以试试，之后可以跟我反馈哦~

2. 项目根目录下的hamster.sql，将其中的SQL语句导入MySQL中执行，创建好数据库和数据表。

3. 将项目的web目录下的所有文件复制到tomcat的webapps目录下的某个文件夹内（这个文件夹将会影响你的contextPath值），并进入这个文件夹，这个文件夹在这里称为根目录。

4. 接着进入根目录下的WEB-INF/classes下：

>* 修改jdbc.properties文件如下：
找到该部分，修改你的项目数据库配置（数据库驱动、连接url、帐号、密码），如果你不用mysql也行，请自行导入相应数据库驱动jar包，并设置好相应参数即可：
```
driver=com.mysql.jdbc.Driver
url=jdbc:mysql://localhost:3306/myblog?useUnicode=true&characterEncoding=utf8
user=root
password=
```

>* 修改config.properties文件如下：
```
# solr搜索引擎地址
solr.url=http://127.0.0.1:8080/solr/rss
# 本机地址
server.host=http://localhost:8080
# 注册功能开关，true的话黑客自行注册一个账号就可能破坏你的数据哦
user.register=false
# 管理员邮箱
user.email=1098129797@qq.com
# 启用动态主页
index.dynamic=true
# 开发者模式免登录，true的话是开发期间免登录方便使用的，上线之后肯定要关掉的
mode.dev=false
# 图片上传地址
uploadUrl.image.ckeditor=/upload/images
# 全局缓存开关
global.cache.enable=false
```
solr.url：为solr全文搜索引擎地址，暂时还未实现（主要是似乎没什么必要。。。）  
server.host：为邮件内容涉及到你的服务器地址，在这里配置好可直接在邮件内容中点击超链接进入部署的网站  
user.register：为网站管理员注册功能开关，首次使用配置为true，自行注册一个管理员帐户之后设置为false，防止恶意用户进入后台糟蹋。。。  
user.email：为游客留言等自动通知管理员的目标电子邮件地址
index.dynamic：动态主页的动态方式，false就是动态主页也采用静态主页的方式。tips：静态主页网址为/index.html，动态网页网址/index/
mode.dev：开发者免登录模式开关
uploadUrl.image.ckeditor：图片上传路径
global.cache.enable：全局缓存开关

>* logback.xml配置日志级别

>* WEB-INF目录下的web.xml文件：
配置错误页面和默认主页（其中主页设置有两种模式）：
>>* 静态主页：
```
    <welcome-file-list>
        <welcome-file>index.html</welcome-file>
    </welcome-file-list>
```
>>* 动态主页：
```
    <welcome-file-list>
        <welcome-file>index.jsp</welcome-file>
    </welcome-file-list>
```
>>* index.html：静态化主页，每次后台修改相应的文章它会自动更新，并且不进入后台它也会每隔半小时自动更新一次。  
index.jsp：通过SpringMVC进入的动态主页，得到的页面数据一定是当前最新的。  

>* WEB-INF/classes下的MailServer.properties（设置所有邮件通知的发件方Email服务器、帐号密码等一系列设置）：  
```
    #----------------\u8FD9\u4E24\u4E2A\u662F\u6784\u5EFAsession\u5FC5\u987B\u7684\u5B57\u6BB5----------
    #smtp\u670D\u52A1\u5668
    mail.smtp.host=smtp.mxhichina.com.[邮件服务商smtp主机地址]
    #mail.smtp.host=smtp.163.com
    #\u8EAB\u4EFD\u9A8C\u8BC1
    mail.smtp.auth=true[认证]
```
